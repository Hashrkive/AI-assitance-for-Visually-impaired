# AI Assistant for the Visually Impaired

**Empowering independence through intelligent voice-guided navigation and real-time environmental awareness.**

## 🧠 Overview

This project develops an intelligent AI-powered voice assistant specifically designed for visually impaired individuals. The system leverages computer vision, natural language processing, and real-time audio feedback to transform daily navigation and task completion into seamless, independent experiences — without relying on expensive hardware.

## ✨ Features

- 🎙️ **Voice Command Recognition** — Captures and interprets natural user commands using noise-canceling speech input
- 👁️ **Real-Time Object Detection** — Identifies obstacles, hazards, and points of interest using YOLO / MobileNet
- 🗺️ **Voice-Guided Navigation** — Spatial audio cues and TTS feedback for safe indoor/outdoor movement
- 💬 **Natural Language Understanding** — Contextual NLP for intuitive, conversational interaction
- 📳 **Haptic Feedback Module** — Non-audio tactile alerts for notifications and navigation cues
- 📱 **Affordable & Device-Agnostic** — Runs on standard smartphones/tablets using open-source frameworks

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Language | Python 3.9+ |
| Computer Vision | OpenCV, TensorFlow / PyTorch, YOLO, Mask R-CNN |
| Speech Recognition | Google Cloud Speech API, CMU Pocketsphinx |
| Text-to-Speech | gTTS / Pyttsx3 |
| NLP | BERT, GPT-based Transformer models |
| Mobile Deployment | TensorFlow Lite |
